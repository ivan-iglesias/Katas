## FizzBuzz

Se elige un número "n" del 1 al 100. Crearemos una cadena de texto del 1 a "n"  en la que reemplazaremos los números multiplos de 3 por Fizz, los multiplos de 5 por Buzz y los multiples de ambos por FizzBuzz. De no ser multiplo de ninguno se dejara el número tal cual.

Siendo ```n = 15```

```
1, 2, Fizz, 4, Buzz, Fizz, 7, 8, Fizz, Buzz, 11, Fizz, 13, 14, FizzBuzz
```

Para el ejercicio solo se permiten numeros del 1 al 100, de elegir otro número se generara una excepcion del tipo *ArgumentException*.

## Fibonaci

La sucesión de Fibonacci comienza con los números 0 y 1, siendo los siguientes terminos la suma de los dos anteriores.

```
n = 0 -> 0
n = 1 -> 1
n = 7 -> 13 (0, 1, 1, 2, 3, 5, 8, 13)
```

Para el ejercicio solo se permiten numeros del 0 al 100, de elegir otro número se generara una excepcion del tipo *ArgumentException*.
